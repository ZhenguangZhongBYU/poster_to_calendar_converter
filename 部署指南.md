## 🚀 部署步骤

### 1. 创建GitHub仓库

1. 登录 [GitHub](https://github.com)
2. 点击右上角 "+" → "New repository"
3. 填写信息：
   - **Repository name**: `anqi-time-manager`
   - **Description**: `安琪时间管理大师 - 海报识别工具`
   - **Public** ✅
   - **Add README** ✅
4. 点击 "Create repository"

### 2. 上传文件

#### 方法A：网页上传（推荐）

1. 进入刚创建的仓库
2. 点击 "uploading an existing file"
3. 拖拽以下文件到上传区域：
   - `index.html`
   - `README.md`
   - `.gitignore`
4. 填写提交信息：`Initial commit: Add simplified time management tool`
5. 点击 "Commit changes"

#### 方法B：Git命令

```bash
# 克隆仓库
git clone https://github.com/你的用户名/anqi-time-manager.git
cd anqi-time-manager

# 复制文件到仓库目录
# 将 index.html, README.md, .gitignore 复制到当前目录

# 提交并推送
git add .
git commit -m "Initial commit: Add simplified time management tool"
git push origin main
```

### 3. 启用GitHub Pages

1. 进入仓库 → **Settings**
2. 左侧菜单 → **Pages**
3. **Source** 选择：
   - "Deploy from a branch"
   - Branch: **main**
   - Folder: **/ (root)**
4. 点击 **Save**

### 4. 访问网站

等待2-3分钟，访问：
`https://你的用户名.github.io/anqi-time-manager`
