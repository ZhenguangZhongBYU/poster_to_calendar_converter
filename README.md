# 🎉 安琪时间管理大师 - GitHub Pages版本

一个轻量级的海报识别工具，可以拍照识别海报中的事件信息并自动生成Google Calendar事件。

## ✨ 功能特点

- 📸 **多种输入方式**：支持文件上传、拖拽上传和直接拍照
- 🤖 **AI智能识别**：使用豆包API识别海报中的事件信息
- 📅 **自动格式化**：将识别结果转换为标准的事件格式
- 🕐 **时间智能推断**：结合WorldTimeAPI获取当前时间信息
- 📥 **一键下载**：生成ICS文件，可直接导入Google Calendar
- 📱 **移动端友好**：响应式设计，支持手机端使用
- 🌐 **GitHub Pages部署**：无需服务器，直接部署到GitHub Pages

## 🚀 部署到GitHub Pages

### 方法一：直接上传文件

1. **创建GitHub仓库**
   - 登录GitHub，创建新仓库
   - 仓库名建议：`time-management-master` 或 `anqi-time-manager`

2. **上传文件**
   - 将 `index.html` 文件上传到仓库根目录
   - 将 `README.md` 文件也上传到仓库

3. **启用GitHub Pages**
   - 进入仓库设置 (Settings)
   - 找到 "Pages" 选项
   - 选择 "Deploy from a branch"
   - 选择 "main" 分支和 "/ (root)" 文件夹
   - 点击 "Save"

4. **访问网站**
   - 等待几分钟后，访问：`https://你的用户名.github.io/仓库名`

### 方法二：使用Git命令

```bash
# 1. 克隆仓库
git clone https://github.com/你的用户名/仓库名.git
cd 仓库名

# 2. 复制文件
# 将 index.html 和 README.md 复制到仓库目录

# 3. 提交并推送
git add .
git commit -m "Initial commit: Add time management tool"
git push origin main

# 4. 启用GitHub Pages（在GitHub网站上操作）
```

## 📱 使用方法

1. **打开网站**
   - 在浏览器中访问你的GitHub Pages地址
   - 或直接打开 `index.html` 文件

2. **配置API**
   - 在页面顶部输入你的豆包API Key
   - 确认接入点ID正确

3. **上传图片**
   - 点击上传区域选择文件
   - 或拖拽图片到上传区域
   - 或点击"打开相机"直接拍照

4. **识别事件**
   - 点击"开始识别事件"按钮
   - 等待AI分析图片内容

5. **下载日历**
   - 查看识别结果
   - 点击"下载ICS文件"
   - 导入到Google Calendar等应用

## 🐛 故障排除

### 常见问题

1. **API调用失败**
   - 检查API Key是否正确
   - 确认网络连接正常
   - 查看浏览器控制台错误信息

2. **图片识别不准确**
   - 确保图片清晰，文字可读
   - 尝试调整图片角度或光线
   - 检查海报是否包含完整的事件信息

3. **相机无法使用**
   - 检查浏览器权限设置
   - 确保使用HTTPS协议
   - 尝试使用文件上传方式
